# TACN Profiles

Generated TACN profile scaffolding from RadioReference TXT export.

- `tacn-all/` contains:
- `control_channels.txt` (all TACN control-capable channels from the TXT)
- `talkgroups.csv` (all TACN talkgroups)
- `talkgroups_with_group.csv` (same rows with RR group labels)

Metadata:
- `system.json`
- `sites.json`
